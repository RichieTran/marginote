# Marginote

Annotate and track highlights across the web, organized by project and subgroup.

## Load the unpacked extension

1. Open Chrome and navigate to `chrome://extensions`.
2. Toggle **Developer mode** on (top-right).
3. Click **Load unpacked** and select this project's root folder
   (the folder containing `manifest.json`).
4. Pin **Marginote** from the extensions menu for easy access.

## Using it

- Click the toolbar icon to open the popup. From there you can pick the
  active project and click **Open Dashboard**.
- Press **Ctrl+Shift+M** (macOS: **Cmd+Shift+M**) from any tab to jump
  straight to the dashboard.

## Development notes

- No build step: the codebase is vanilla ES modules, HTML, and CSS. Edit
  a file, then hit **Reload** on the Marginote card in `chrome://extensions`.
- Icons are generated by `assets/generate-icons.mjs` using Node built-ins
  only; re-run that script if you change the palette.
- Conventions live in `CONVENTIONS.md`. Manual verification notes live in
  `TESTING.md` and should be updated after each prompt's work lands.
